5.部署

将生成好的博客部署到github上

5.1 新键仓库

登录github(https://github.com/)，如果没有账号的话先注册账号。

新键仓库，名称需要为 xxx.github.io ；这里的xxx跟你的用户名要保持一致，填好之后滑动到页面最下方点击的 create repository按钮创建

5.2 设置token

在同步的博客到github仓库时，需要在github中设置token，用户名的方式在2021年后已经不让用了

点右上角自己头像展开下拉菜单 —> Settings

进入设置页面之后，滑倒页面最底部，左侧菜单中选择Developer settings

进入页面后，左侧菜单展开Personal access tokens —> 选择Tokens

点击右上角的按钮Generate new token —> 选择第二个 Generate new token(classic)

填写描述文字，选择有效日期，勾选权限，一般勾选两个就够了repo 和 delete repo。

滑动到页面最下方，点击Generate token 按钮，生成一个token,点击复制按钮将生成的token保存到记事本，后面配置博客时会用到
